---

## Ação

Implemente um programa que recebe um número inteiro e imprima SIM se ele é maior ou igual a 0.

### Entrada

- Um inteiro

### Saída

- "SIM" se o número for maior ou igual a zero, caso contrário, apenas uma quebra de linha.

## Exemplos

```txt
>>>>>>>> 0
3
========
SIM
<<<<<<<<

>>>>>>>> 1
-12
========

<<<<<<<<

>>>>>>>> 2
-3
========

<<<<<<<<

>>>>>>>> 3
1
========
SIM
<<<<<<<<

>>>>>>>> 4
0
========
SIM
<<<<<<<<
```
