![Imagem eh Positivo](https://raw.githubusercontent.com/qxcodefup/arcade/master/base/maximo/cover.jpg)

## Ação

Implemente um programa que recebe dois números inteiros e imprime o maior.

### Entrada

- Dois inteiros

### Saída

- Imprima o maior número
